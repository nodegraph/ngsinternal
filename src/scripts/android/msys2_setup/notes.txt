In your msys2 .bashrc file, simply source the setup_env.sh script in this directory.
This script will read the $ARCH env variable to bring up an appropriate environment.

You will like want to create a desktop shortcuts similar to the following:

%comspec% /k ""C:\Users\shing\Documents\nodegraphsoftware\src\scripts\android\msys2_setup\launch_msys2.bat" ARCH_WINDOWS"
%comspec% /k ""C:\Users\shing\Documents\nodegraphsoftware\src\scripts\android\msys2_setup\launch_msys2.bat" ARCH_WINRT"
%comspec% /k ""C:\Users\shing\Documents\nodegraphsoftware\src\scripts\android\msys2_setup\launch_msys2.bat" ARCH_ANDROID"

Note that these shortcuts are setting the $ARCH env variable for setup_env.sh to read.
